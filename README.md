# 🌹 Más Allá del Dragón

> *Un RPG narrativo creado con amor, para Sant Jordi.*

**Más Allá del Dragón** es un juego de rol narrativo desarrollado en React con estética retro CRT y estilo pixel-art moderno. Una historia íntima y emotiva sobre valor, ansiedad, crecimiento personal… y una rosa.

---

## 🗺️ Historia

El juego sigue a **Víctor**, un joven que emprende un viaje épico para enfrentarse a un dragón. Pero el verdadero enemigo no es el dragón.

La aventura se desarrolla en **10 capítulos**:

1. **Introducción** — El mundo de Víctor y su punto de partida.
2. **El viaje** — Primeros pasos hacia lo desconocido.
3. **Aliados** — Encuentros que cambian el camino.
4. **La cueva oscura** — Enfrentarse al miedo interior.
5. **El falso final** — La derrota del dragón... ¿o no?
6. **La revelación** — Algo más grande acecha.
7. **El guardián** — El elefante que guarda la verdad.
8. **Recordar a Marta** — La acción más poderosa del juego.
9. **El desenlace** — Una rosa, un libro, un mensaje.
10. **Final feliz** — Crecimiento, amor y esperanza.

---

## ⚔️ Mecánicas

- **Sistema de estado de Víctor**: HP, MP, Ansiedad y Percepción.
- **Combate por turnos** con decisiones narrativas que importan.
- **Jefe final**: El elefante guardián, con una debilidad especial.
- **Acción especial**: *"Recordar a Marta"* — el ataque más poderoso del juego.
- **Desenlace emocional**: rosa 🌹, libro 📖 y mensaje final de crecimiento.

---

## 🛠️ Tecnologías

- [React](https://react.dev/) + Vite
- Estética retro **CRT** con scanlines y efectos de fósforo
- Pixel-art moderno para sprites y escenarios
- JavaScript (ES2024)

## 🚀 Desarrollo local

```bash
# Usa Node 20
nvm use 20

# Instala dependencias
npm install

# Arranca el servidor de desarrollo
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
