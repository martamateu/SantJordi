# Más Allá del Dragón

RPG narrativo en React con estética retro CRT y estilo pixel-art moderno.

Incluye:

- Historia por 10 capítulos (intro, viaje, falso final del dragón, revelación y final feliz)
- Sistema básico de estado de Victor (HP, MP, ansiedad y percepción)
- Combate final contra el elefante guardián con acción especial **“Recordar a Marta”**
- Desenlace emocional con rosa, libro y mensaje final de crecimiento

## Desarrollo

```bash
nvm use 20
npm install
npm run dev
```

## Validación

```bash
npm run lint
npm run build
```
